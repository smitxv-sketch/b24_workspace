## Золотые правила (не нарушать)

1. **Переменные БП** — только `$this->SetVariable()`, никаких `{=Variable:x} = ...`
2. **URL с кириллицей** — `rawurlencode()` для **каждой части пути** отдельно
3. **Подпапки** — `Folder::getList()`, никаких `getChildren()` без параметров
4. **UF-поля в БП** — `BpStorage::readField()`, а не `Factory::getItem()` (кэш!)
5. **Права** — код доступа всегда с префиксом: `U123`, `G5`, `DR1`
6. **Задачи** — `CTaskItem($id, 1)`, не `CTasks::GetByID()` (не работает в БП)
7. **Файлы задач** — только через `SERVICE_DATA = 'TASK_RESULT'`, не `UF_TASK_WEBDAV_FILES`
8. **Логирование** — `BpLog::registerFatalHandler()` в начале каждого sys_*.php
9. **Контекст** — `BpContext::detect($this)` вместо ручного разбора GetDocumentId()
10. **Конфиг** — `BpConfigValidator::validate()` в master_init.php перед стартом БП
