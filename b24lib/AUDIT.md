## ⚠️ Пропущено — нужно добавить в новые файлы (3 места)

### Пропуск 1 — `01_BP.md`: неправильное имя переменной

Из `Запись_данных_в_переменные_бизнес-процессов_Битрикс24.md`:

> Имя переменной БП должно быть только латиницей и подчёркиваниями.  
> При кириллице или пробелах — значение пишется без ошибки, но теряется.

Добавить в `01_BP.md §1.5`:
```php
// ❌ Кириллица и пробелы в имени — значение потеряется
$this->SetVariable('Имя папки', $value);

// ✅ Только латиница и подчёркивания
$this->SetVariable('folder_name', $value);

// ❌ Переменная создана в коде, но не создана в настройках БП
// Код выполнится без ошибки, но значение исчезнет
$this->SetVariable('non_existent_var', $value);
```

### Пропуск 3 — `03_CRM.md`: раздел учёта времени неполный

Из `Учёт_времени_и_отчёты_в_Битрикс24.md` не попало:

- SQL за **произвольный период** (с конкретными датами `'2025-01-01'`)
- SQL **детализация по задачам** (с `INNER JOIN b_tasks`)

- **BI-конструктор отчётов** — путь настройки SQL-источника данных в интерфейсе
- **Ежемесячный отчёт через БП** (автоматическая рассылка)
